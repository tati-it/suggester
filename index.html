C

C counts as a general-purpose language, but in practice, it is mainly used for specific fields: systems and embedded programming.It is used in cases where there are no clear alternatives, and you should choose to learn this language when you need to build these types of applications.

JavaScript

It’s impossible to be a software developer these days without using JavaScript in some way. According to Stack Overflow's 2019 Developer Survey, JavaScript is the most popular language among developers for the seventh year in a row.

Along with HTML and CSS, JavaScript is essential to front-end web development. A majority of the web’s most popular sites, from Facebook and Twitter to Gmail and YouTube, rely on JavaScript to create interactive web pages and dynamically display content to users.

Because JavaScript has a forgiving, flexible syntax and works across all major browsers, it is one of the friendliest programming languages for beginners. JavaScript is primarily a front-end language run on the browser, it can also be used on the server-side through Node.js to build scalable network applications.

Ruby

A dynamic, open source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write.
